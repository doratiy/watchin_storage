# Пульт охраны банка
Это внутренний репозиторий для сотрудников банка «Синие». Если вы попали в этот репозиторий случайно, то вы не сможете его запустить, т. к. у вас нет доступа к БД, но можете свободно использовать код верстки или посмотреть как реализованы запросы к БД.
Пульт охраны — это сайт, который можно подключить к удаленной базе данных с визитками и карточками пропуска сотрудников нашего банка.

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

`pip install -r requirements.txt`

## Как запустить

### Для работы программы создайте файл .env и впишите в него следующее

```
ENGINE = 'django.db.backends.postgresql_psycopg2'

DB_HOST = ''

DB_PORT= ''

DB_NAME = ''

DB_USER = ''

DB_PASSWORD = ''
```

Названия переменных говорят о том что они должны хранить.

Для запуска пропишите в терминал ```python main.py```


Пример ответа программы:


`Количество пропусков: 100
Активных пропусков: 89`


`passcards = Passcard.objects.all()
active_pascards = Passcard.objects.filter(is_active=True)`

`passcard` - переменная, получающая все значения из базы данных
`active_passcards` - переменная, получающая фильтрованные значения `is_active=True`, то есть значения True от активных пропусков.
